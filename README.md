# e-plantShopping

## Project Name: Paradise Nursery

### Overview
Paradise Nursery is a front-end React application designed for a seamless houseplant shopping experience. This application allows users to browse a curated selection of plants, view detailed product information, and manage their selections through an interactive shopping cart. 

### Features
* **Interactive Shopping Cart:** Powered by Redux, allowing users to add items, adjust quantities, and remove plants with real-time updates to the total cost and cart badge.
* **Product Catalog:** Displays at least six unique houseplants organized into distinct categories, complete with thumbnails and pricing.
* **Seamless Navigation:** Utilizes React Router to navigate between a welcoming landing page, the main product listing, and the detailed shopping cart view.

### Technologies Used
* React.js
* React Router DOM
* Redux Toolkit
* CSS
